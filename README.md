# Mini Catch Game 🎮

A simple **HTML animation game** where the player catches a falling ball using a movable platform. The game is built using **HTML, CSS, and JavaScript** in a single file.
### Link :- https://box-move-mu.vercel.app/

## 📌 Features

* Simple and fun gameplay
* Mouse-controlled player movement
* Real-time score system
* Smooth animation using `requestAnimationFrame`
* No external libraries required

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript

## 🎮 How to Play

1. Open the `game.html` file in your browser.
2. Move your **mouse left or right** inside the game area.
3. Catch the falling ball with the platform.
4. Each successful catch increases your **score**.
